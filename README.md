# gopeg-mv4
Source code to combine mp4 with FFmpeg golang source code

## Install 
```shell
go get github.com/Watson-Sei/gopeg-mv4
```

## Build Test Environment
```shell
$ make test_input
# add video sample provided video 
$ make test_output
$ go test 
```
